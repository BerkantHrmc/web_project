# Film DTO Klasörü

## Amaç

Film modülü için Data Transfer Object'ler (DTO) tanımlarını içerir.

## Dosyalar

- **create-film.dto.ts**: Film oluşturma için DTO
- **update-film.dto.ts**: Film güncelleme için DTO

## Özellikler

- `class-validator` decorator'ları ile validasyon
- TypeScript tip güvenliği
- Otomatik validasyon (ValidationPipe ile)
