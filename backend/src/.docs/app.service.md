# app.service.ts

## Amaç

Uygulamanın ana servisidir. Basit iş mantığı fonksiyonları içerir.

## Temel Bileşenler

### `AppService` Sınıfı

- `@Injectable()`: Dependency injection için işaretlenmiş

### Metodlar

#### `getHello(): string`
- **Açıklama**: "Hello World!" string'ini döndürür
- **Kullanım**: Test amaçlı basit bir endpoint için
