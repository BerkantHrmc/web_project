# Contexts Klasörü

## Amaç

React Context API kullanılarak global state yönetimi için context'leri içerir.

## Dosyalar

- **AuthContext.jsx**: Authentication context (login, register, logout, user state)

## Özellikler

- Global state yönetimi
- Authentication state
- Token yönetimi
- Axios header yapılandırması
